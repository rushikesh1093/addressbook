# Address Book System

Python Address Book with UC1–UC15: contacts, multiple books, search, count, sort, and file I/O (TXT, CSV, JSON).

## Run

```bash
cd address_book_project
python main.py
```

## Features

- Create, add, edit, delete contacts (UC1–UC5)
- Multiple address books (UC6)
- Duplicate check by name (UC7)
- Search by city/state (UC8)
- Count by city/state (UC10)
- Sort by name or city (UC11, UC12)
- Save/load: TXT, CSV, JSON (UC13–UC15)
